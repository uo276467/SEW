¿Qué es Embers.js?

Embers.js es un framework web de código abierto en JavaScript que utiliza un patrón de componente-servicio.

Se usa en sitios web tales como Apple Music, Square Inc, Groupon, LinkedIn, Live Nation o Twitch entre muchos otros.

Filisofía y diseño

Según la empresa, Embers se diseñó en torno a cuatro ideas clave: centrarse en aplicaciones web ambiciosas, productividad desde el primer momento, estabilidad sin estancamiento y previsión de futuros estándares web.
